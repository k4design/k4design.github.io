## Description Options

### Option 1: Comprehensive

**Real Estate Autofill** makes it easy to create and update property listing designs in Figma. Instead of manually editing each text layer and image, tag your layers once and let the plugin do the work.

**Key Features:**

• **Tag-Based System**: Tag any text or image layer with predefined property fields (bedrooms, bathrooms, price, address, features, agent info, and more)

• **Bulk Updates**: Update ALL tagged layers across your page with a single click—no need to select anything

• **CSV Import/Export**: Upload property data via CSV files for rapid batch processing. Includes a downloadable template

• **Image Management**: Upload and manage up to 10 property images with drag-and-drop support. Images are automatically applied to tagged layers

• **Data Refresh**: Extract existing data from your tagged layers to populate the form, making edits quick and easy

• **Property-Specific Fields**: Pre-configured fields for real estate listings including bedrooms, bathrooms, square footage, price, address, city/state, tagline, description, features (1-6), agent name/title/phone/email

Perfect for real estate agents, property managers, and designers who create multiple listing designs and need a fast, consistent workflow.

### Option 2: Concise

### Option 3: Feature-Focused

Create and update property listings faster with **Real Estate Autofill**. This plugin uses a tag-based system to automatically populate your designs with property data and images.

**How it works:**
1. Tag your text and image layers (e.g., "price", "bedrooms", "img:1", "img:logo")
2. Enter data manually or upload a CSV file
3. Click "Apply" to update all tagged layers instantly

**What you get:**
• 20+ pre-configured property fields (bedrooms, bathrooms, sqft, price, address, features, agent info, etc.)
• Up to 10 image slots per listing plus logo support
• CSV template for bulk data entry
• One-click updates for entire pages
• Data extraction from existing layers

Ideal for real estate professionals who design multiple listings and need consistency and speed.
