How to install

Gmail
  1. Click the gear icon → See all settings.
  2. Under “General”, scroll to “Signature”.
  3. Create or select a signature, then paste your signature in the box (use Copy Rich Signature for best results).
  4. Save changes.
Outlook desktop
  1. File → Options → Mail → Signatures.
  2. Create or edit a signature.
  3. Paste your HTML (paste into the editor; Outlook will accept the formatting). Use Copy Rich Signature or paste from the HTML code if needed.
  4. OK to save.
Outlook on the web
  1. Click the gear icon → View all Outlook settings.
  2. Mail → Compose and reply → Email signature.
  3. Paste your signature (use Copy Rich Signature).
  4. Save.
Apple Mail
  1. Mail → Settings (or Preferences) → Signatures.
  2. Select an account and create or edit a signature.
  3. Paste your signature. Use Copy Rich Signature so formatting is preserved.
