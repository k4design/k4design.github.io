Magazine

Tag the template once. Pour a hundred pages in one pass, for every listing after.

A single-property magazine is dozens of text boxes and photo frames laid out beautifully — and re-typed by hand for every new listing. Dezzy ends the re-typing. You tag each placeholder once, then fill the whole issue from a form and a folder of photos.

Part A Do once per template

Tag the template

Let the bulk tools do the heavy lifting, then fix up by hand. When you're done, save the template — every future listing is a duplicate, and the tags come along for free.

1

Auto-number the photo boxes

Bulk → Auto-number photo slots

Walks your page frames in reading order — Cover_Outside, Insert Page 1, and so on — finds the placeholder rectangles, and tags them photo.slot.001, photo.slot.002… Two matching boxes across facing pages are detected as a spread and share one number.

2

Scan layer names into tags

Bulk → Scan layer names → tags

Any well-named layer tags itself. This is the reward for naming your template layers sensibly.

  • A text layer named price, description, feature1, caption.1.body → the matching field.
  • Any layer named #something → a custom tag (text layer becomes text, image frame becomes image).
Bulk tab · Tagging
🏷️
Scan layer names
Turns #field names and schema-named layers (street, price, description…) into tags. Idempotent — already-tagged layers are skipped.
Scan
Click Scan — well-named layers tag themselves in one pass.
3

Categorize the photo frames

Generic slots work, but categories are the magic — a "kitchen" photo lands in a kitchen frame automatically. A frame named Master Bath 2 becomes photo.masterbath.02; or on the Tag tab pick "Kitchen slot (next #)" from the Photos picker.

Known categories: entry, kitchen, dining, living, master bed, master bath, secondary bed, secondary bath, office, amenities, garage, outdoor, exterior, floor plan, interior.

4

Tag anything left over, by hand

Tag tab — select layer(s) on the canvas, then use the picker that fits:

  • Content (text) — Property, Features, Agent, Captions, or a custom key.
  • Photos (image) — named slots (cover, hero, agent headshot), category slots, full-page, or a custom key.

Each picker has its own Assign button, so the tag type is never ambiguous.

Couple spreads & set crops

One image bleeding across two facing frames? Select both and click Couple 2 as spread ⇔ — they share one photo and pin to opposite edges, reading as one continuous picture with no seam down the gutter.

Any image tag can pin its crop with the Align toggle — ◄ Left · Center · Right ►.

Lock what must never change

In the Tag picker, under Locks: mark permanent artwork with photo.locked and boilerplate or disclaimers with text.locked. Locked layers are skipped everywhere and reported as "N locked" rather than flagged empty.

Part B Repeat per property

Build a listing

Duplicate the template, open the plugin, and move through Content → Photos → Apply.

1

Enter the content

Content tab — fill the form: Property, Features, Agent, Captions. Values autosave into the file as you type. Each field shows how many layers carry it; untagged means a value there won't land anywhere — a hint you missed a tag in Part A.

Shortcut: Download a field,value CSV as a fill-in template for coordinators, then Import it back in bulk.

5

Apply

Apply all text pours text into every tagged layer. Apply photos cover-fits each assigned photo, renders coupled spreads seamlessly, and fills any unassigned slot solid magenta so empty placeholders are impossible to miss. Each apply is one Undo step and can be cancelled mid-run.

Reference

Worth knowing

Portability
Save state (.zip)

Bundle every field value, photo, and assignment into one archive on the Bulk tab — then restore it in another file. A complete, portable listing.

Fix-up
Retag from images

Photos already placed but tags wrong? Bulk reads each frame's image, identifies it by filename, and re-tags to the right category — after a preview.

Full bleed
Full-page slots

Big full-bleed images are fed room-by-room rather than by filename, so a shoot flows through the magazine in a natural order.

Persistence
The image vault

An invisible off-canvas rectangle pins your uploaded photos' bytes in the file so Figma won't discard them. Don't delete 🗄 Dezzy image vault.

When something's off

Troubleshooting

SymptomCause & fix
A value didn't show upThe field is untagged — check the count on the Content tab, then tag the layer.
A frame came out magentaIts slot was unassigned — pick a photo on the Photos tab and re-apply.
Some text didn't applyThe layer is text-locked, or its font is missing (check the apply report).
A spread has a seamSpread drift — the halves changed size. Run Check magazine, then Align spread fills.
A thumbnail is blankPhoto lost before the vault existed — re-upload the same file and it re-links automatically.
Gallery empty on reopenNormal — thumbnails rebuild themselves as the plugin pulls bytes back from Figma.
