User Guide

Dezzy Magazine

Turn any layout into a data-driven property magazine — tag once, then fill text and photos in seconds.

What it does

Dezzy Magazine turns an existing Figma layout into a reusable, data-driven template. You tag each text layer and photo container once; after that you enter listing details, upload the photo shoot, and the plugin places everything — headlines, captions, prices, and dozens of photos — exactly where it belongs.

Everything runs locally in your file. There is no network access, no account, and nothing is uploaded to a server.

The five tabs

  • Tag
  • Content
  • Photos
  • Apply
  • Bulk

Open the plugin on your template page and work left to right: tag the layout, enter content, load photos, then apply.

Quick start

  1. Tag your templateOn the Tag tab, run Scan layer names to auto-tag layers you've named after fields and rooms, then Auto-number photo slots to turn the rest of the photo containers into numbered slots. You can also select any layer and tag it by hand.
  2. Enter the listingOn the Content tab, fill in the grouped fields (address, price, description, captions, agent…). Values autosave into the file. Or import a CSV.
  3. Load the photosOn the Photos tab, drop in the shoot. Click Auto-assign to route photos into slots by filename, or set any slot by hand.
  4. ApplyUse the fixed Apply all content / Apply photos bars, or go to the Apply tab to validate first and apply everything in one pass.

Tagging

Name-based scan

Name a text layer after a field (price, #tagline) or a photo layer after a room (Kitchen, Master Bath 2) and the scan tags it automatically. First matching category wins, so "master bathroom" counts as Master Bath, not Secondary. Layers that match nothing are left alone — sweep those into generic slots with Auto-number.

Tags travel with the file

Tags and entered content are stored in the Figma file itself, so they survive duplication. Set the template up once and reuse it for every new listing.

Partial (template) text

You can tag only part of a text layer — swap the dynamic words while leaving boilerplate untouched — using template tokens like {{price}}.

Content

The Content tab is a clean, grouped form (Property, Features, Agent, Captions). It autosaves as you type. A fixed Apply all content bar sits at the bottom of the tab.

  • CSV import/export. Download a fill-in field,value CSV, complete it in a spreadsheet, and import it back — handy for handing data entry to someone else.
  • Locked layers. Mark boilerplate or disclaimers as locked so Apply never overwrites them.

Photos

Print quality preserved

Upload 50–100 photos at once; each is processed one at a time to keep memory flat. Images are only downscaled if they exceed Figma's 4096 px limit — anything that already fits is stored untouched. Placement uses cover-fit cropping (never stretched), and you can pin the crop left, center, or right per container.

One photo, many slots

A photo can be assigned to as many slots as you like — just pick it in each slot's dropdown. The gallery badge shows every slot a photo lands in.

How auto-assign routes photos

Name photos after rooms and the plugin sends them to matching slots, in filename order within each category.

Coupled spreads

Couple two containers on facing pages and a single photo tiles seamlessly across both — the crop windows are computed so the image reads as one continuous shot across the gutter. Use Align spread to make the two halves line up perfectly.

Apply & validate

  • Pre-flight check. The Apply tab flags empty fields, unassigned slots, and mismatched spreads before you commit.
  • Apply text, photos, or everything in one pass, with a progress bar and cancel.
  • Magenta placeholders. Any slot left unassigned fills magenta after Apply, so nothing ships half-finished.

Save state (Bulk tab)

Export a single .zip that bundles your entered content, photo assignments, every layer tag, and the image files themselves — for backup or hand-off. Import it into a fresh copy of the template to restore all of it. Tags re-apply immediately; content and photos wait until you Apply.
